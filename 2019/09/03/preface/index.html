Welcome 欢迎

This blog site serves as a notebook where I’ll keep adding stuff while studying computer science and technology. I’ll try to compose each blog post in a bilingual style: English and Chinese (they are not exactly translated from each other). The content of this blog will be including but not limited to:

此博客是本人为学习计算机科学技术所建立，用于收集相关资料和记录知识点。 我会尝试使用中英双语写作（两者互相并非精准翻译）。博客的内容包括但不限于：

  • Tips and skills in learning Linux : As a computer guy, Windows is not my first option. Learning Linux may give you a hard time at the beginning, but I believe it is totally worth it after you get familiar with it.

  • Notes and summaries for computer science literature : I’ll briefly write a summary if I find any interesting or important academic paper in the field of computer science. I think this is a good practice on independent studying. Meanwhile, it will help me on collecting papers for my dissertation’s bibliography section.

  • Techniques on coding : Solving coding problems is THE key to entering the computer industry. I hate it but there is no other way around. Beside the algorithm problems, I’ll also include tutorials about useful developing tools and languages.

  • Notes on interesting research topics : There are several topics I’m personally interested in: machine learning, artificial intelligence, natural language processing, distributed systems, networks and more. I hope this can be a place to store the knowledge from different research areas.

  • Notes in learning English : As an English-as-a-second-language speaker, I always think that continuously practicing English is as significant as coding. It is also beneficial to studying computer science. A coder can not talk is worth nothing.

  • Comments on world news and more : Always keep an eye on what is going on in the real world. Don’t isolate yourself from the outside.

About this blog 关于此博客

This blog is not only a warehouse of knowledge, but also a practice of web development and text edit. The content is written in Markdown, which is easy to use and maintain. Currently, this blog is still under construction.

此博客不仅用来储存知识，同时也是网站开发和文本编辑的练习。内容使用Markdown编写，易于使用和维护。博客目前正在完善中。
